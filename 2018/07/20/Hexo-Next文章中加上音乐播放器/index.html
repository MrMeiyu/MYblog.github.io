Hexo Next文章中加上音乐播放器

发表于 2018-07-20 | 分类于 技术文章

首先需要了解一下Aplayer

下载地址：GitHub Aplayer。
下载到本地以后，解压后把dist文件夹复制到themes/next/source文件夹下。

music.js

新建一个js，路径为themes/next/source/dist/images/music.js;
内容为：

const ap = new APlayer({
container: document.getElementById('aplayer'),
fixed: true,
autoplay: false,
audio: [
  {
    name: "往后余生",
    artist: "王贰浪",
    url: "http://music.163.com/#/m/song?id=571338279",
    cover: "",
  }
]

如果还需要其他的参数，请参考：Aplayer中文文档
